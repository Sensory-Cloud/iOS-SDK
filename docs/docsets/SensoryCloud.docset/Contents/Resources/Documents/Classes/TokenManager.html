TokenManager

public class TokenManager
extension TokenManager: CredentialProvider

A token manager class that may be used for managing and securely storing OAuth credentials for Sensory Cloud.

Once credentials have been generated, TokenManager will automatically provide access tokens to the Sensory Cloud services See the documentation for CredentialProvider for instructions on providing your own token management for Sensory Cloud
